# 🎓 Conclusion Générale

## Bilan du projet

Le projet **Senebus Live** a été réalisé dans le cadre du stage de fin d'études pour l'obtention du diplôme de **Technicien Supérieur en Développement Web et Mobile**.

L'objectif principal était de concevoir une application mobile capable d'améliorer l'accès aux informations relatives aux transports en commun en proposant une interface moderne, intuitive et performante.

Au terme du développement, les objectifs fixés ont été atteints. L'application permet notamment aux utilisateurs de consulter les lignes de bus, les horaires, les arrêts, de gérer leurs favoris et de disposer d'un espace personnel sécurisé grâce à Firebase Authentication.

Ce projet a également permis de mettre en pratique les connaissances acquises durant la formation, notamment en développement mobile avec Flutter et en utilisation des services cloud de Firebase.

---

# 🎯 Objectifs atteints

Les principaux objectifs définis au début du projet ont été réalisés :

* développement d'une application mobile multiplateforme avec Flutter ;
* intégration de Firebase Authentication pour la gestion des utilisateurs ;
* utilisation de Cloud Firestore comme base de données ;
* stockage des images avec Firebase Storage ;
* mise en place d'une architecture modulaire facilitant la maintenance ;
* développement d'une interface utilisateur simple et intuitive ;
* utilisation de Git et GitHub pour le suivi des versions ;
* rédaction d'une documentation technique complète.

Ces réalisations démontrent la faisabilité et la pertinence du projet.

# 🌱 Perspectives d'amélioration

Bien que l'application soit fonctionnelle, plusieurs améliorations pourront être apportées dans les prochaines versions.

## Géolocalisation

Intégrer la localisation en temps réel afin d'afficher la position des bus sur une carte interactive.

---

## Notifications

Ajouter un système de notifications permettant d'informer les utilisateurs de l'arrivée prochaine d'un bus ou d'éventuelles perturbations.

---

## Paiement électronique

Permettre l'achat de tickets directement depuis l'application grâce à des solutions de paiement sécurisées.

---

## Historique

Mettre en place un historique des recherches et des trajets favoris afin de faciliter les consultations futures.

---

## Mode hors connexion

Ajouter un mode hors ligne permettant de consulter les dernières informations synchronisées même sans connexion Internet.

Proposer plusieurs langues afin de rendre l'application accessible à un plus grand nombre d'utilisateurs.
